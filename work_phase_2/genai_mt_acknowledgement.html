Reflections on Generative AI Use

Incorporating GenAI into Web Design and Development

In DECO7140, I used Generative AI (GenAI), like ChatGPT and GeminiAI at most of my times, to enhance my web design process, balancing creativity, technical tasks, and ethics. For creativity, GenAI I asked for analogies from real life that simplified concepts like external CSS and JavaScript, showing better file organization. Technically, it explained JavaScript, helping me build a temperature widget with the fetch() method—though I verified its suggestions using W3Schools. I documented some of GenAI’s role and avoided directly using its outputs as mine. W3Schools website played an all round role in enhancing coding and learning which I preferred more over AI use.

  • Creativity: Use GenAI for initial ideas (e.g., layouts or features), adjusting to project needs.
  • Technical Aid: Consult GenAI for syntax or troubleshooting or correcting any minor error unseen from naked eyes
  • Ethics: Acknowledge GenAI use and assess its outputs in way of learning.

GenAI was a valuable tool, but I prioritized my learning and understanding. I avoided reliance by cross-checking information with Wc3Schools webiste and using it as a guide. This approach ensured I developed understanding first and then benefiting from GenAI’s capabilities.

Critical Engagement with GenAI and MT Tools in Design Process

In DECO7140, I critically applied GenAI to my design process. While user research wasn’t emphasized early on, GenAI could suggest personas or questions—requiring real user data to validate. It clarified Progressive Web Apps and JavaScript Promises, but shallow responses needed cross-checking with MDN Web Docs and tutors. I avoided dependency by coding and aksing tutord about the temperature widget independently after using GenAI for guidance. For future projects:

  • Ideas: Start with GenAI concepts, refining via feedback.
  • Research: Test GenAI for research tools, using outputs in real data from public access websites .
  • Problem-Solving: Use GenAI to break tasks into steps and continue learning using real life analogies

GenAI helped improve productivity and understanding, yet critical evaluation and independent work ensured authentic learning and ethical practice.
