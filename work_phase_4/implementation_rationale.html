Implementation Rationale

  • 1. Buttons have been implemented with the hovering effect using the accent colour upon hovering to keep the interface engaging and interactive. This approach not only enhances the visual appeal but also provides users with clear minimal design good for visualization and simplicity.

  • 2. The hovering over the cards was changed to the root colour scheme for consistency and visualization

  • 3. Submit buttons have been styled to match the accent color scheme instead of primary, ensuring they stand out and are easily identifiable.

  • 4. For Events page, initially an accordion layout was considered to organize event information effectively, but it was later changed to a grid layout with info-cards as some errors kept on repeating making it hard to trace.

Our approach focuses on accessibility, ensuring that all members can benefit from the features offered. We prioritize responsive design, ensuring compatibility across devices, and implement best practices in web development to guarantee a seamless experience.
